# Hoja de Vida

Este repositorio contiene una versión web de una hoja de vida personal (CV) diseñada para mostrarse en el navegador. Incluye HTML, CSS y recursos estáticos.

## Contenido
- Página principal: `src/index.html` — estructura y contenido de la hoja de vida.
- Estilos: `src/index.css` — estilos y diseño.
- Assets: `assets/` — imágenes y otros recursos usados en la página.

## Estructura del proyecto
- src/
  - index.html — HTML de la hoja de vida.
  - index.css — estilos.
- assets/ — imágenes y archivos estáticos.

## Cómo ver la hoja de vida
1. Abrir [src/index.html](src/index.html) en tu navegador (doble clic o "Abrir con..." desde tu editor).
2. Para desarrollo, puedes usar una extensión de servidor local en VS Code (Live Server) o servir la carpeta con cualquier servidor estático.

## Cómo editar
- Modifica el contenido (nombre, experiencia, educación) en [src/index.html](src/index.html).
- Cambia estilos en [src/index.css](src/index.css).
- Añade o reemplaza imágenes en [assets/](assets/).

## Notas
- Mantén una copia de seguridad antes de grandes cambios.
- Si deseas exportar a PDF, imprime la página desde el navegador (Ajusta estilos para impresión si es necesario).

## Contacto
Actualiza la sección de contacto en [src/index.html](src/index.html) con tu información personal.
